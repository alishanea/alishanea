AWS Architecture — SnapLink

A fully serverless, scalable URL shortener built as a B.Tech Final Year Cloud Computing Project. Uses 7 AWS services in a production-grade architecture.

🚀

AWS Lambda

4 functions: Shorten, Redirect, Analytics, List. Auto-scales to 0 when idle.

🔌

API Gateway

REST API with Cognito auth, rate limiting (50 RPS), CORS, and request validation.

🗄️

DynamoDB

Two tables: URLs and Analytics. PAY_PER_REQUEST billing + TTL auto-expiry.

🪣

S3

Hosts the static frontend. Private bucket — only accessible via CloudFront.

🌐

CloudFront

Global CDN with HTTPS, HTTP/2, origin access control, and SPA routing support.

🔐

Cognito

User pool with email auth, JWT tokens, and API Gateway authorizer integration.

📊

CloudWatch

Custom dashboard, Lambda error alarms, P99 latency alerts, and SNS notifications.

🔍

X-Ray

Distributed tracing across Lambda and API Gateway for performance profiling.

Project Highlights for Resume

    Cost Optimized — Serverless PAY_PER_REQUEST billing. Estimated cost: ~$0/month at low traffic on AWS Free Tier
  • ✓
    Observability — CloudWatch dashboard + alarms + X-Ray tracing = production-grade monitoring
